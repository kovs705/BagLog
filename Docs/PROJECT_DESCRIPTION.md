# BagLog — описание проекта

## Граница Release 1.0

Первый релиз — local-first приложение, проверяющее основной цикл работы с
наборами: создать → добавить вещи → опубликовать локально → fork → изменить
fork. «Публикация» в этом релизе показывает набор только в каталоге на том же
устройстве; аккаунтов, сети и общего публичного каталога пока нет.

Полный обязательный объём, критерии готовности и отложенные возможности
описаны в [Release 1.0](RELEASE_1_0.md).

## Краткое описание

BagLog — это социальная платформа для создания, публикации и форка практичных списков вещей для повседневной жизни.

Проект вдохновлен EDC-культурой, форматом “what’s in my backpack” и идеей GitHub-like форков. Пользователи могут публиковать свои наборы вещей: everyday carry, содержимое рюкзака, городское вело-снаряжение, тревожные наборы, travel packing lists, фото/видео сетапы, рабочие комплекты, походные наборы и другие практичные списки под конкретные сценарии.

Главная идея продукта — не просто показать “что у меня есть”, а дать другим людям возможность скопировать список, адаптировать его под себя и опубликовать свою версию с указанием оригинального источника.

## Позиционирование

BagLog — это “GitHub для реальных наборов вещей”: часть EDC-витрина, часть личная база снаряжения, часть recommendation network и часть community knowledge base.

Платформа помогает людям находить, сравнивать, копировать и адаптировать практичные списки вещей вместо того, чтобы собирать рекомендации по Reddit, YouTube, блогам, заметкам, таблицам и разрозненным постам в соцсетях.

## Ключевой сценарий

1. Пользователь создает список под конкретный сценарий.
2. Добавляет описание, вещи, категории, ссылки и фотографии. (some features requires IAP)
3. Публикует список в общий каталог (free for all).
4. Другой пользователь находит этот список.
5. Нажимает Fork.
6. Получает копию списка.
7. Меняет вещи под свой бюджет, страну, стиль жизни или задачу.
8. Публикует свою версию.
9. На странице новой версии отображается ссылка на оригинальный список.

## Главная ценность

BagLog помогает пользователям быстро понять, какие вещи нужны для конкретного сценария, посмотреть реальные наборы других людей и адаптировать готовый список под себя.

Ценность не только в публикации, а в повторном использовании: хороший список становится шаблоном, который другие люди могут улучшать, менять и развивать.

## Отличие от обычной соцсети

Обычная соцсеть показывает посты. BagLog показывает структурированные списки, которые можно использовать.

Главное отличие:

- не просто фото;
- не просто текстовый пост;
- не просто рекомендация;
- а список, который можно скопировать, изменить и применить.

## Главная гипотеза

Люди захотят не только смотреть чужие EDC / backpack / gear lists, но и копировать их, адаптировать под свои задачи и публиковать свои версии.
